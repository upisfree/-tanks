# How to build your own −tanks?
 1. Install [node.js & npm](https://github.com/joyent/node/wiki/Installation)
 2. Install [grunt](http://gruntjs.com/getting-started)
 3. 
```bash
git clone https://github.com/upisfree/-tanks.git
cd ./-tanks
npm install
grunt
```
